# Ravi and Sam Wedding Website Setup

## Step-by-Step Guide

1. **Install Dependencies**
   ```sh
   npm install
2. **Update guest list**
   ```sh
   node guestListHash.js
3. **Compile sass to css, minify js, etc.**
   ```sh
   gulp
4. **Launch locally compiled webpage in your browser**
   ```sh
   firefox index.html
